# FinTrack - Personal Finance Tracker

A clean and minimalistic web app to help you track your personal income and expenses.

<img width="648" height="582" alt="Screenshot 2026-07-02 at 11 28 45 PM" src="https://github.com/user-attachments/assets/5c78fa29-8c17-4f2c-8891-3345e62dc900" />


## ✨ Features

- Add income and expense transactions
- Real-time dashboard with Total Balance, Income, and Expenses
- Interactive pie chart showing spending by category
- Filter transactions (All / Income / Expense)
- Delete transactions
- All data automatically saved using `localStorage` (persists after refresh)
- Fully responsive design (works great on mobile and desktop)
- Clean, modern, and easy-to-use interface

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Chart.js (for data visualization)

1. Clone the repository
2. Open `index.html` in your browser
3. No installation needed — it works completely offline
